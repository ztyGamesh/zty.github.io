万物皆对象

“万物皆对象”核心在于如何去理解”对象”这个概念

tips: 当然，也不是所有概念都是对象，值类型就不是对象。

来看一个重要的常用运算符–typeof.

typeof函数输出的所有类型值，在此列出:

function typeofShow() {
    console.log(typeof x); //undefined
    console.log(typeof 1); //number
    console.log(typeof 'abc'); //string
    console.log(typeof true); //boolean

    console.log(typeof function () {}); //function

    console.log(typeof [1, 2, 3]); //object
    console.log(typeof {a: 1, b: 2}); //object
    console.log(typeof null); //object
    console.log(typeof new Number(1)); //object
}

typeofShow();

上面代码列出了typeof输出的所有类型标识，分为两个概念。

值类型

  • undefined
  • number
  • string
  • boolean

引用类型

  • object
  • function (也是object)

也就是说 函数、数组、对象、null、new Number(1)都是对象。它们都是引用类型

判断一个变量是不是对象非常简单。

- 值类型: typeof  
- 引用类型: instanceof

var fn = function () {};
console.log(fn instanceof Object); // true

进入正题，在javascript中，对象到低是如何定义的呢

对象–若干属性的集合

java或者C#中的对象都是new一个class出来的，而且里面有字段、属性、方法，规定的非常严格。

javascript的ES6语法中也有了class的概念，不过原生还是基于new Function()的原理，只是一个语法糖。

在javascript中，对象的概念是比较”随意”的，数组是对象，函数是对象，对象肯定还是对象。

对象里面的一切都是属性，理解的时候可以理解成只有属性，所谓的方法其实也是属性的一种，只有属性，没有方法。

属性表示为: 键值对 的形式

更加灵活的是，javascript中的喜爱那个可以任意的扩展属性，没有class的约束。

placeholder

以上代码中，obj是一个自定义的对象，其中a、b、c就是它的属性，而且在c的属性值又是一个对象，它又有两个属性。

这个传统的对象可能比较好理解，那么函数和数组也可以定义属性么。当然可以，只要是对象，它就是属性的集合。

var fn = function () {
    console.log('我也是对象')
}
fn.a = 888;
fn.b = function () {
    console.log('我是一个属性')
}
fn.c = {
    name: 'zty',
    year: 1993
}

上段代码中，函数就作为对象被赋值了a、b、c三个属性，很明显，这就是属性的集合。

question: 函数作为对象使用的使用场景能举个例子吗?

answer: 看看jQuery

在jQuery源码中，`jQuery` or `$`, 这个变量其实是一个函数  

console.log(typeof $) // function  
console.log($.trim(" aaa ")); // "aaa"

通过typeof验证，$确实是个函数。$.trim也是个函数。

很明显，这就是在$函数上加了一个trim属性，属性值是函数，作用是截取字符串前后空格.

本节重点

一切(引用类型)都是对象，对象是属性的集合

函数和对象的关系

上节已经提到， 函数就是对象的一种，因为通过instanceof函数可以判断。

var fn = function () {};
console.log(fn instanceof Object); // true

对！函数是一种对象，但是函数却不像数组一样–你可以说数组是对象的一种，因为数组就像是对象的一个子集一样。但是函数与对象之间，却不仅仅是一种包含和被包含的关系，函数和对象之间的关系比较复杂，甚至有一种鸡生蛋蛋生鸡的逻辑。这一节来看看这个问题。

function Fn() {
    this.name = "zty";
    this.year = 1993;
}
var fn1 = new Fn();

上面的这个例子很简单，它能说明: 对象可以通过函数创建, 对，也只能说明这一点。

本节的核心思想:

对象都是通过函数创建的

有人可能说不对:

var obj = {a: 1, b: 2};
var arr = [1, 2, 3];

这里obj和arr就是字面量生成的呀。

但是，这里所谓的”字面量”其实是一种—-“快捷方式”，在编程语言中，一般叫做”语法糖”。

// 其实上面代码的本质是  

var obj = new Object();
obj.a = 1;
obj.b = 2;

var arr = new Array();
arr[0] = 1;
arr[1] = 2;
arr[2] = 3;

而其中的Object和Array都是函数:

console.log(typeof Object); // function  
console.log(typeof Array); // function

所以: 对象都是通过函数来创建的

结论:

  1. 对象都是函数创建的
  2. 函数是对象 那么 函数 and 对象 究竟是什么关系呢？
